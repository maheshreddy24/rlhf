# RLHF

> **Note:** This repository is currently under active development.
>
> Its goal is to provide implementations for training language models using Reinforcement Learning from Human Feedback (RLHF), supporting both PPO (Proximal Policy Optimization) and GRPO (Group Relative Policy Optimization) methods on GPT-2 (124M parameters).

The repository currently includes implementations for:

- Supervised Fine-Tuning (SFT)
- Reward Model Training
- RLHF Training Pipeline

Additional features, improvements, and documentation are being added as development progresses.
